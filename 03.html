모델 설명

LSTM + ResNet-50 + Sequence

파이프라인: 입력 이미지 시퀀스는 ResNet-50을 통해 특징이 추출되고, 이 특징들은 LSTM 네트워크에 전달되어 시퀀스 데이터를 처리합니다. 최종적으로 위치와 방향을 예측하는 fully connected layer로 연결됩니다.

아키텍처: ResNet-50은 깊은 CNN으로 이미지 특징을 추출하고, LSTM은 시퀀스 특징을 학습합니다. 최종 레이어는 위치와 쿼터니언(회전)을 예측합니다.

Poseur

파이프라인: Poseur 모델은 고유의 네트워크 아키텍처를 사용하여 입력 이미지로부터 직접 위치와 방향을 예측합니다.

아키텍처: Poseur는 CNN 기반의 특징 추출 네트워크와 fully connected layer로 구성되어 있으며, 입력 이미지로부터 위치와 쿼터니언을 예측합니다.
