📌 CAMERA POSE REGRESSION

가설 1.

손실 함수로 Huber Loss를 사용하면 MSE 보다 안정적인 loss 그래프를 출력할 것이다.

sequence로 입력한 train data set에 존재할 outlier에 robust한 손실 함수 필요.

→ MSE와 MAE의 장점을 취합한 Huber Loss 사용

Huber Loss

L1과 L2의 장점을 취하면서 단점을 보완하기 위해 제안된 것으로 모든 지점에서 미분이 가능하면서 이상치에 로버스트하다.

Smooth L1 loss

실제 모델에서는 Huber loss와 그래프 형태가 유사한 Smooth L1 loss를 사용하였다.

Huber loss와 Smooth L1 loss 비교

가설 2.

PoseNet의 Euclidean Loss를 사용하면 Huber Loss 보다 loss 감소율이 높을 것이다.

6-DoF pose를 사용하여 regression을 진행하는 PoseNet은 위치 손실과 방향 손실을 합한 값을 loss function으로 사용한다.

→ camera pose regression과 변수 종류가 같으므로 성능 향상에 유용할 것.

x는 위치 벡터, q는 quaternion 벡터에 대한 손실 값이다.

β는 위치 손실과 방향 손실에 대한 비중을 결정하는 가중치이다. 위치와 방향의 크기가 다르기 때문에 β를 적절히 조절하여 두 손실의 균형을 맞춘다.
